# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — through issues, pull requests, discussions, or any other interaction — you agree to abide by it.

Concerns about behavior in this community may be reported privately by emailing the maintainers (address to be published before the public launch). All reports are reviewed promptly and confidentially.
